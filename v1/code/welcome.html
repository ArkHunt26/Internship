So let's see a small example for blinking the in-built LED in Arduino UNO R3.

practice

Description
The following project is to blink the in-built LED that is pin 13 to glow ON for 1 second and 1 second OFF.

Flow-chart
Coding

void setup()
{
 pinMode(LED_BUILTIN, OUTPUT);
}

void loop()
{
 digitalWrite(LED_BUILTIN, HIGH);
 delay(1000);
 digitalWrite(LED_BUILTIN, LOW);
 delay(1000);
}

Description

  1. In setup() set the pins as INPUT or OUTPUT. In this case set in-built LED pin (pin 13) set as OUTPUT. This is done using pinMode().Syntax for this is pinMode(pinNumber, pinState);. This will run only ones
  2. Enter into loop() which will run indefinitely
  3. Make the respective LED ON. This is done by digitalWrite(pinNumber, pinState); function.
  4. Next give delay of 1 second using delay(milliseconds).
  5. Make the respective LED OFF. This is done by digitalWrite(pinNumber, pinState); function.
  6. Next give delay of 1 second using delay(milliseconds).

So let's see a small example for blinking the in-built LED in PIC.

practice

Description
The following project is to blink an LED that is to glow ON for 1 second and 1 second OFF.

Flow-chart
Coding

#pragma config FOSC = HS
#pragma config WDTE = OFF
#pragma config PWRTE = OFF
#pragma config BOREN = OFF
#pragma config LVP = OFF
#pragma config CPD = OFF
#pragma config WRT = OFF
#pragma config CP = OFF

#include <xc.h>
#define _XTAL_FREQ 20000000

void main(void) {
  TRISB=0x00;
  PORTB=0x00;
  while(1)
  {
    PORTB=0xFF;
    __delay_ms(1000);
    PORTB=0x00;
    __delay_ms(1000);
  }
  return;
}

Description

  1. In First set the div Configuration Bits for PIC16F877A.
    • FOSC: It is used to set the Oscillator Selection Bit. In this we can select the clock source. To know more check in datasheet or Microship developer Help
    • WDTE: It is used to set the Watchdog Timer Enable bit. To know more check in datasheet or Microship developer Help
    • PWRTE: It is used to set the Power-up Timer Enable bit. To know more check in datasheet or Microship developer Help
    • BOREN: It is used to set the Brown-out Reset Enable bit. To know more check in datasheet or Microship developer Help
    • LVP: It is used to set the Low-Voltage (Single-Supply) In-Circuit Serial Programming Enable bit. To know more check in datasheet or Microship developer Help
    • CPD: It is used to set the Data EEPROM Memory Code Protection bit. To know more check in datasheet or Microship developer Help
    • WRT: It is used to set the Flash Program Memory Write Enable bits. To know more check in datasheet or Microship developer Help
    • CP: It is used to set the Flash Program Memory Code Protection bit. To know more check in datasheet or Microship developer Help
  2. Next lets include xc.h file by adding #include <xc.h>.
  3. Next lets add clock frequency by defining #define _XTAL_FREQ 20000000. The _XTAL_FREQ already exsists in builtins.h file.
  4. Next let's create a main() since while compiling a program with standard C library, the entry point of your program is set to _start. Now that has a reference to main() function internally. So, at compilation time, there is no (need for) checking the presence of main(). At linking time, linker should be able to locate one instance of main() which it can link to. That way, main() will serve as the entry point to your program. Check in stackoverflow.
  5. Next let's set PortB as output that is by TRISB = 0x00;. This will make Port B as OUTPUT. And let us set Port B to be LOW initially by using PORTB=0x00;.
  6. Next like add a while loop so that the blink code runs indefinitely.
  7. Now let's make Port B HIGH in the beginning of the loop by using PORTB=0xFF;
  8. Next let's add a delay of 1 second by using __delay(milliseconds); function. In place of milliseconds we will give a value. In this case we want to make LED high for 1 second so we give a value of 1000.
  9. Now let's make Port B LOW in the beginning of the loop by using PORTB=0x00;
  10. Next let's add a delay of 1 second by using __delay(milliseconds); function. In place of milliseconds we will give a value. In this case we want to make LED high for 1 second so we give a value of 1000.

Description
The following project is to blink an LED that is to glow ON for 1 second and 1 second OFF. Click here to download STM32 procedure.

Flow-chart
Coding

while (1)
{
  /* USER CODE END WHILE */

  /* USER CODE BEGIN 3 */
  HAL_GPIO_WritePin(GPIOB, GPIO_PIN_7, GPIO_PIN_SET);
   HAL_Delay(3000);
  HAL_GPIO_WritePin(GPIOB, GPIO_PIN_7, GPIO_PIN_RESET);
  HAL_Delay(1000);
}

Description

  1. Follow the procedure to setup in the IOC file.
  2. Next scroll to main() paste the code at the respective positions as shown.
  3. In while() loop we can see the HAL_Drivers being used.
  4. To make the pin high uses HAL_GPIO_WrtirePin(PortName, PortNumber, PinState);.
  5. Next let's keep a delay using HAL_Delay(milliseconds); and give 1000 as the value to have 1 second delay.
  6. To make the pin low uses HAL_GPIO_WrtirePin(PortName, PortNumber, PinState);.
  7. Next let's keep a delay using HAL_Delay(milliseconds); and give 1000 as the value to have 1 second delay.
